# PWA Icons

Bu klasörde PWA için gerekli icon dosyaları bulunmalıdır.

## Gerekli Icon Boyutları

- `icon-72x72.png` - 72x72px
- `icon-96x96.png` - 96x96px
- `icon-128x128.png` - 128x128px
- `icon-144x144.png` - 144x144px
- `icon-152x152.png` - 152x152px (iOS)
- `icon-192x192.png` - 192x192px (Android)
- `icon-384x384.png` - 384x384px
- `icon-512x512.png` - 512x512px (Splash screen)

## Icon Oluşturma

1. Ana logo dosyanızı (`fog_logo_pink.png`) kullanarak iconlar oluşturun
2. Tüm iconlar PNG formatında olmalıdır
3. Iconlar maskable olmalıdır (kenarlarda padding olmalı)
4. Arka plan şeffaf veya solid renk olabilir
